## Server / Serveur

### Installation / Installation

```bash
$ npm install
```

### Launching Server / Lancement du Serveur

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Running Server Tests / Exécution des Tests du Serveur

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
